This creates new (blank) timecards for the current payroll week, for *ALL* active productions. It will also create timecards for the following week IF the current date is within the number of days specified by the payroll preference for that production. For each production, timecards are created for all employees that have valid (completed) Start Forms with effective dates that include one or more days within the specified week. An Event log entry is created when this completes, showing the number of timecards created for each production.
Ignore prior week's timecard status (not currently allowed)
