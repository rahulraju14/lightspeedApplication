Production Elements

Production Element List

The Production Element List is the complete list of cast members, locations, and things considered or used to play script elements in the production.  A production element, such as a location, can be used for many different sets and in the case of a TV series, can be used in multiple episodes.  Production elements are not specific to any episode or project.  They span the entire production.

 

The list displays each element’s name, its start and end date for the current production or episode, and the element category.

 

As with script elements there are 21 categories of production elements.  Category names are identical to those used for script elements with two exceptions.  Script element Characters are related to production Cast members and script elements Sets are related to production Locations.  You can filter the element list using the category dropdown at the top of the list.

 

Show Production Elements linked to this Project only

This checkbox is located above the Production Element List and is present if you are working in a TV series with multiple episodes.  Select this checkbox if you wish to only display production elements which are associated with script elements in your current episode.  Otherwise, all production elements are displayed.

 

Creating a New Production Element

Use the following steps to create a new production element:

 

1.      Select the Add button at the top of the Production Element list.

The Add Production Element popup window appears.

 

2.      Select an element category from the dropdown and click Add.

 

Note: It is not possible to create a new cast member via the Production Element screen.  Cast members are created via the Cast & Crew page under the Cast & Crew tab.

 

3.      Complete the following fields for the new element:

a.      Enter a Name

b.      Enter a Description

c.      Enter any Special Instructions

d.      Select a Manager or Site Rep

The Manager is typically the crew member or vendor responsible for representing, providing, or safeguarding the production element.  For example, the Prop Master may be responsible for safeguarding an antique rifle used by the production.

Locations are assigned Site Reps.

e.      Add an association with a script element using the Associated with/Playing table.

                                                              i.      Click the Associate button

A list of script elements from the current production or episode appears.

                                                            ii.      Select a script element from the list and click on the + sign next to that element

                                                          iii.      Select the Close button

                                                          iv.      Using the Status dropdown in the Played By table, choose a status for the association – Selected, Not Selected, or Candidate.

f.        Edit the Blackout Dates data

                                                              i.      Add a blackout date period using the Add Date button

                                                            ii.      Remove a blackout date period by clicking on the red x next to the applicable date range

g.      Select the Images mini-tab and add images, if applicable

 

4.      Select the Save button in the lower right to save your changes.

Production Element View

The right pane displays the details of a selected production element.

 

Viewing a Production Element

Select a production element from the list to view its details on the right.  With the exception of Locations, the production element screen contains two mini-tabs: Details and Images.  Location elements include two additional mini-tabs: Map and Manage.

Details

The Details mini-tab displays the information for the selected production element.  This information includes the script elements associated with the production element and any blackout dates when the production element is unavailable.

Images

The Images mini-tab displays images that have been uploaded for the element.

Map (Locations only)

The Map mini-tab allows you to upload a map for the location and enter instructions for driving directions and parking.

Manage (Locations only)

The Manage mini-tab is intended for use by Location Managers.  This screen allows you to take notes on permits and to associate Points of Interest with the location.  Points of Interest that you assign to the location will appear on the location report.

 

Editing a Production Element

Select the Edit button in the upper right corner of the screen to edit the selected production element.  The production element screen switches to edit mode allowing you to edit the element data as needed.  Note that the element category is not editable.

 

While in edit mode you can perform the following tasks:

Select a Manager or Site Rep

The Manager is typically the crew member or vendor responsible for representing, providing, or safeguarding the production element.  For example, the Prop Master may be responsible for safeguarding an antique rifle used by the production.

 

Locations are assigned Site Reps.

Associate the Element with One or More Script Elements

Using the Associated with/Playing table you can associate the production element to one or more script elements.  This association means that the production element is a candidate to play the script element in the production.  For example, you can associate a location with a set if you'd like to consider that location as a candidate to represent the set during the production.

 

1.      Click the Associate button.

 

2.      Select a script element from the list and click on the + sign next to that element.

 

3.      Select the Close button.

 

4.      Using the Status dropdown in the Playing table, choose a status for the association – Selected, Not Selected, or Candidate.

 

5.      Select the Save button to save your changes.

 
Remove an Association from the Playing Table

 

1.      Locate the script element or role from the table and click on the red x icon next to that element.

 

2.      Select the Save button to save your changes.

 

Add a Blackout Date

You can add a date or range of blackout dates using the Blackout Dates table.  A blackout day is a day that the production element is not available.

 

1.      Select the Add Date button.

The Add Blackout Dates popup window appears.

 

2.      Select a Start Date, End Date, and Reason.

 

3.      Click on the Save button.

 

4.      Select the Save button to save your changes.

 

Remove a Blackout Date

To remove a blackout date entry from a production element, locate the entry in the Blackout Dates table and click on the red x next to the applicable entry.  Confirm the deletion and then select Save.

Add an Image of the Production Element

 

1.      Select the Images mini-tab.

 

2.      Click on the New Image button.

The Add New Image popup window appears.

 

3.      Use the controls to select and upload an image file.

The image is added to the production element record.

 

4.      Select Save to save the changes.

 

Delete an Image of the Production Element

 

1.      Select the Images mini-tab.

 

2.      Locate the image you want to delete and click on the Delete Image button.

The Delete Image popup window appears prompting you to confirm.

 

3.      Click the Delete button to confirm.

The image is removed from the production element record.

 

4.      Select Save to save the changes.

 

Add a Map to a Location (Locations only)

 

1.      Click on the Edit button if you are not already in edit mode.

 

2.      Select the Map mini-tab.

 

3.      Click on the Add/Create Map button.

 

4.      Use the controls to select and upload a map image file.

 

5.      Select Save to save the changes.

 

Remove a Map from a Location (Locations only)

 

1.      Click on the Edit button if you are not already in edit mode.

 

2.      Select the Map mini-tab.

 

3.      Click on red x icon at the lower left corner of the map image.

 

4.      At the confirmation popup, select Delete.

 

5.      Select Save to save the changes.

 

Add a Point of Interest to a Location (Locations only)

 

1.      Click on the Edit button if you are not already in edit mode.

 

2.      Select the Manage mini-tab.

 

3.      Click on the Add button in the Points of Interest table.

The Add Point of Interest to Location popup window appears.

 

4.      Points of Interest which have already been created are displayed in a list.  To add a point of interest to your location click on the + sign next to the point of interest.

The point of interest is added to your location.

 

Note: You may add up to four points of interest to a location.

 

5.      Select Close to close the window.

 

6.      Select Save to save the changes.

 

Remove a Point of Interest from a Location (Locations only)

 

1.      Click on the Edit button if you are not already in edit mode.

 

2.      Select the Manage mini-tab.

 

3.      Within the Points of Interest table, click on red x icon next to the point of interest you wish to remove.

 

4.      Select Save to save the changes.

 

Deleting a Production Element

Select the Delete button in the upper right corner of the screen to delete the selected production element.  The Delete Production Element popup window prompts you to confirm.  Click on the Delete button to delete the production element.

 

Note: Cast production elements cannot be deleted via the method described above.  To delete a cast member use the Cast & Crew screen on the Cast & Crew tab.

 

Warning: Once a production element has been deleted, it cannot be restored.
