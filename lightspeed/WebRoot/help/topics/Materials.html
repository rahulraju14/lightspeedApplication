Materials

The Materials screen is used to track film stock inventory and other media used on the production.  Select the Materials link under the Project tab to view your media inventory.

Materials List

On the left side of the screen is the Materials List.  Each media type that you enter and track is listed here.

 

Creating a New Media Type

Use the following steps to create a new media type for your production.

 

1.      Select the Add button at the top of the Materials List.

The Add Media Type popup window appears.

 

2.      Enter the name of the new media type.  Click Save.

The media type is added to the list and an inventory entry is automatically created with today's date in the right pane.

 

3.      Click on the inventory entry to edit any of the fields.

 

4.      Select Save to save the changes.

Materials Inventory Screen

The right pane is used to display the inventory for each media type.

 

Viewing Media Inventory

Select a media type from the list on the left to view its inventory on the right.  Individual inventory line items are listed in the right pane ordered by date with the most recent date at the top.  Each line item represents a change in inventory for a given day, and tracks the following amounts:

 

·         Received

Raw film stock or media that is purchased and added to inventory.

 

·         Print

Good film stock or media that was successfully used for the production.

 

·         No Good

Film stock or media that was found to be no good.

 

·         Waste

Film stock or media that was discarded for various reasons.

 

·         Remaining

Raw film stock or media remaining in inventory.

 

At the bottom of the page are totals for Received, Print/Good, No Good and Waste.  The most recent tally of Remaining media is listed here as well.

 

Editing Media Inventory

From the view screen, select the Edit button to edit the inventory entries.

Include Inventory on Production Report

If you want the media inventory for this media type to appear on the production report in the Media Usage section, select the Include on Production Report checkbox.  If you do not want this media type to appear on the production report, uncheck the checkbox.

Edit an Inventory Line Item

Click on a line item to edit it. 

Add an Inventory Line Item

To add a new inventory line item, select the Add Entry button at the upper right corner of the table.  Select a date for the inventory entry from the popup window and click on Save.  Input the inventory amounts for the new entry.

Delete an Inventory Line Item

To delete an entry, click on the red x icon next to the entry's date.

 

When you've completed making your changes select the Save button in the lower right corner of the screen.

 

Deleting a Media Type

If you wish to completely remove a media type, including its entire inventory history, from the system, select the media type in the left hand list and select the Delete button in the upper right corner of the screen.  A popup will ask you to confirm the deletion.

 

Warning: Once a media type has been deleted, it cannot be restored.
