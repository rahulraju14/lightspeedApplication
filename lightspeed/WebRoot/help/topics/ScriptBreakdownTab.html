Script/Breakdown Tab

The Script/Breakdown tab displays the scripts, scenes, and elements for the production.  If you are working on a TV series then the scripts and script elements will be for the specific episode in which you are working.

 

For a given production or episode multiple script revisions can be uploaded.  One of these revisions, however, must be selected as the Revision in Use.  The script revision in use is the one that is used by the breakdown pages, strip boards, and shooting schedule.  You can change which script revision is the one in use at any time.

 

If you hover the mouse on the Script/Breakdown tab the first two submenu selections are Breakdown and Script.  The remaining menu options allow you to view Script Elements, Production Elements, and Points of Interest.

 

·         Script Elements are people, places, and things in a script’s scenes.  For example, if a scene consists of two people in a parking lot hiding behind a car, the people, parking lot, and car are all Script Elements.

Script Elements are the elements listed on each scene’s breakdown page.

 

·         Production Elements are the real life people, places, and things used to play the Script Elements during production.  In the above example, the Production Elements for the scene would be the two actors playing the people, the location being used to represent the parking lot, and the rented vehicle used for the car.

 

·         Points of Interest (POI) are businesses and facilities near shooting locations that provide services useful to the production.  Photocopy centers, restaurants, parking facilities, and hospitals are examples of points of interest.  You can associate points of interest with shooting locations, which will cause the points of interest to appear on location reports and directions.
