Strip Board

This page displays the strip board currently in use for the production or episode.

 

The strips listed are those from the current breakdown of the current script revision.  Selecting a different script revision to be the one in use or changing the breakdown by merging or splitting scenes will change the available strips in the strip board.

 

Strips are color coded using the same industry standard scene type colors used on the breakdown page.

Strip Board View

The strip board view screen displays the production's strip boards.  Each production or episode can have multiple strip boards, but only one can be selected as the strip board actively in use.  If you have multiple strip boards and wish to quickly switch between them you can do so using the Strip Board dropdown menu above the strip board.

 

Note: If you have multiple sets of strip boards you can view the full list of strip board sets on the Strip Board Sets screen.

 

Two strip board layouts are available, thick or thin, selectable by the Layout dropdown above the strip board.  In the thick view each strip includes the synopsis information.  The thin view omits this allowing for more strips to be viewed at one time.

 

The strip board view displays the strips in multiple mini-tabs.  By default three mini-tabs are shown - Main Unit Schedule, Additional Units (if applicable), Unscheduled and Boneyard.

 

·         Main Unit: This board makes up the main production shooting schedule.  The order of these strips, broken up by end of day strips, determines which scenes are shot on which production day.

·         Additional Units: Each additional unit in the production or project receives its own board.  You can use this board to create a separate schedule for the additional unit.

·         Unscheduled: These strips represent scenes that are not yet on any of the production schedule boards.

·         Boneyard: These are strips for omitted scenes.  If an omitted scene is restored then the strip is moved to the Unscheduled area.

 

There are three types of strips:

 

·         Scene strips represent one or more scenes and are color coded by scene type.

·         End of Day strips represent a day break in the schedule and are grey.

·         Banner strips are used to present messages to the production staff and are orange.

 

Each Scene strip contains the following detail:

·         The Scene number (or numbers)

·         Whether the scene is Interior or Exterior

·         The Location and Synopsis (synopsis visible in thick layout only)

·         Whether the scene is Day or Night

·         The Script Day

·         The number of script pages for the scene(s)

·         The Cast IDs of the characters appearing in the scenes

 

Each End of Day strip contains the following detail:

·         The words "End of Day" following by the production day number

·         The End of Day's date

·         The total number of script pages planned for that day

 

Banner strips contain text messages.

 

Editing a Strip Board

First view the strip board you want to edit by selecting the appropriate mini-tab.  Only Main unit or additional unit boards may be edited.

 

Select the Edit button in the upper right part of the screen to edit the strip board.  If you have multiple units in your production or episode, first select the mini-tab of the unit you wish to edit.

 

The strip board editor displays two tables of strips.  By default, unscheduled strips are shown on the right and scheduled strips on the left.  The scheduled list can be displayed in both tables by selecting it from the Show dropdown located above the right table.  Showing the scheduled list in both tables is helpful if you are rearranging strips between two completely separate sections of the scheduled list.

 

Use the controls above the right table to filter the strips by INT/EXT or Day/Night, or to sort the strips by any of the column headings.  If you want to include omitted strips in the view, select the Omitted checkbox.

 

Use your mouse to drag and drop strips to create the schedule.  You can hold down the Ctrl or Shift key while selecting strips to select multiple strips at once.

 

To add an End of Day strip, drag and drop the grey End of Day marker strip from above the left table into the list of scheduled strips.  Add a Banner strip by dragging the orange Banner marker strip into the list of scheduled strips.

 

When you are done sorting the strips select the Return button to return to the strip board view.

 

The strip board editor allows you to perform the following tasks:

Strip Board Auto-Scheduler

The Strip board Auto-Scheduler allows you to automatically schedule all of your unscheduled strips by ordering them by up to three criteria - sets, INT/EXT, and Day/Night.  End of Day strips (day breaks) can be inserted after a certain number of strips or pages, or you can distribute the entire schedule over a fixed number of days.

 

Select the Auto-Schedule button in the upper right to use the Auto-Scheduler.  Follow the instructions on the popup window.

 

Warning: The Auto-Scheduler will overwrite the existing scheduled strip board.

Rename the Strip Board

Edit the name of the strip board in the Strip Board Name field.

Omit a Strip

Omitting a strip is similar to omitting a scene on the breakdown page.  Select a strip and click on the Omit button.  The omitted strip is removed from the scheduled or unscheduled list and is moved to the Boneyard.  The scenes on the strip are marked as omitted on the breakdown page.  Omitted strips are shaded in red.

Restore an Omitted Strip

Select an omitted strip and click on the Restore button to restore it.  Restored strips are placed into the unscheduled section of the strip board.  Scenes on the strip are also restored.

 

You can also restore an omitted strip by dragging it into the scheduled table.

Edit a Banner Strip

Select a Banner strip and click on the Edit Banner button.  A popup window allows you to edit the banner text.
