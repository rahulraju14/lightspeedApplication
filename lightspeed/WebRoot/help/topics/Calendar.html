Calendar

Calendar View

The calendar screen displays the production shooting schedule by month.  Above the calendar are scroll buttons that can be used to navigate to adjacent months or years.

 

The legend for the calendar day display is below:

 

·         Production: Production days are represented by a small strip board icon and the list of scenes schedule to be shot that day.  If a call sheet has been published for the day, a call sheet icon is also visible for the day with the crew call time listed.  All icons and scene numbers are linked to their respective data.

·         PRE: Preproduction

·         POST: Postproduction

·         Day off: A red X icon indicates a non-work day

·         Travel: A truck icon represents a company travel day.  An airplane represents other travel

·         Holiday: A holiday or other day off outside the regular schedule

 

If your production or project has multiple units you may view the calendar for a specific unit by selecting it from the Unit dropdown in the upper right.

 

Editing the Calendar

Select the Edit button in the upper right corner of the screen to edit the calendar.  The screen switches to edit mode allowing you to perform the following tasks:

Change the Production Start Date

The production Start Date represents the first day of the production phase.  This is the day on which the schedule from the strip board begins to populate the calendar.

 

Use the Start Date field in the upper left to select a different production start date.  Click on Save to save the change.

Change the Default Weekly Days Off

When in edit mode, check boxes are present above each weekday on the calendar.  These checkboxes represent recurring Days Off for the production.  Place a checkmark in the boxes for the days that will be regular days off.  Click on Save to save any changes.

Modify Day Types

You can change a specific date's day type by dragging and dropping a Day Marker icon into the date.  For example, drag the "Day Off" icon onto a production work day to make that day a day off.  Likewise drag a "Shooting" icon onto a day off to make that day a work day.  Select the Save button to save your changes when you are done.
