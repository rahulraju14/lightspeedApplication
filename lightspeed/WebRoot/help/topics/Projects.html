Projects

The Projects page differs depending on whether or not your production contains multiple projects.  A project is equivalent to an episode in a television series.  Each of your episodes in your series is listed in the project list on the main project page.

 

Feature film productions do not contain multiple projects.

Productions without Multiple Projects (Feature Films, TV Pilots)

Production List

If your production does not have multiple projects the left pane will contain a single entry – the production title.  The right pane (production information pane) displays the details and preferences for this production.

 

Production Information Pane

Two buttons are present along the top of the right pane: Production Members and Edit.

 

·         The Production Members button takes you to a screen where you can view the list of people working on the production.  Use the arrow buttons on the right side of the screen to assign people to various production roles.

 

·         Use the Edit button to switch to edit mode, which allows you to make changes to production data and preferences.  Remember to select the Save button after making your edits.

 

Production Mini-tabs

Details

This mini-tab displays your production title, Production Company information, production start and projected end dates, and the number of people within the cast and crew.  Use the Edit button to edit the Production Company information and projected end date.

 

Production Company

The Production Company name and address will appear in the header of the Call Sheet and daily Production Report.

 

Production Start & End Dates

The production start and end dates cannot be edited on this screen.  To edit the production start date (which represents the first day of principal photography), go to the production Calendar edit page.  The production end date is automatically computed by the production Calendar and strip board.

Preferences

The Preferences mini-tab allows you to set production-wide options.  The available options are:

 

·         Whether or not the production will use automatic notifications.  If you uncheck this option, then Lightspeed will not deliver automatic notifications or documents.

·         Whether or not script page text will be visible online

·         Document watermarking options

·         If Hold & Drop rules will be in effect for cast members, and if so, the default number of hold days used

·         The additional Units used on the production

·         Various task assignments for the production.  Assign responsibility for the following production tasks:

o       Creating & publishing Call Sheets

o       Creating & publishing Production Reports

o       Approving Production Reports

 

Individuals assigned production tasks will receive notifications when those tasks are due or overdue.

Productions with Multiple Projects (TV Series)

If your production consists of multiple projects the left pane is divided into two sections.  At the very top is an entry for the production name.  Below is an area where each of your Projects is listed.

 

Note: If you wish to work in a different project or episode, use the Project dropdown menu above the navigation bar to select a different project.

 

 

Project/Episode List

The Project List displays a list of your projects and the number, title, status, and start date for each.  At the top of the list is the Add button.  Use the Add button to create a new project.  Each of the fields in the list is described below:

 

·         Episode Number

The Episode Number is an alphanumeric code assigned to each Project. Each new Project receives the next incremental number.  If a Project is deleted the number assigned to it is deleted as well.

 

·         Project Title

The title of the Project.  For a TV series, this is the episode title.

 

·         Project Status

Projects are either Active, Read Only, or Archived.

o       Active Projects are ones available for use.  You can log in and use these Projects as you normally would.

o       Read Only Projects are completed.  You can log into these Projects and view their data, but cannot make any changes.

o       Archived Projects are locked and are not accessible.

 

The Default Project

One of the Projects in the Project List is the default Project.  The default Project is the one that users are directed to when they log into the system.  You can change which Project is default using the Make Default checkbox in the right pane.

 

·         Project Start Date

Each Project's start date is displayed in the list.  The start date represents the first day of principal photography.

 

Click on any line item from the project list to select it.  The right pane displays the information for your selection.

Creating a New Project

Use the following steps to create a new project or episode for your production.

 

1.      Create a new Project by selecting the Add button at the top of the Project List.

The Add Project popup window appears.

 

2.      Complete the fields in the popup window.

a.      Enter the name of the new Project.

b.      Select whether or not to copy the project members from the current Project into the new Project.

c.      Select whether or not to copy the Project Preferences from the current Project into the new Project

d.      Select Save when done.

 

3.      The new Project is created and appears in the right pane.  Make any additional changes to the Project as required.

 

4.      Select Save to save the Project.

 

Project Information Pane

The project information pane displays detailed information about the selected project or production.  Along the top of the right pane are screen buttons.  If the production is selected, the Edit button is present.  If a project is selected you will also see the Project members and Delete buttons.

 

·         Project Members takes you to a screen where you can view the list of people working on the specific project.  Each project has its own list of cast and crew members, completely separate from every other project.  Use the arrow buttons on the right side of the screen to assign people to various project roles.

 

·         Use the Edit button to switch to edit mode which allows you to make changes to production/project data and preferences.  Remember to select the Save button after making your edits.

 

·         Use the Delete button to delete the selected project.  Deleting a project deletes all of its scripts, breakdowns, and schedules.

 

Warning: Once a project has been deleted, it cannot be restored.

 

Production/Project Mini-tabs

Details (Production)

If the Production is selected this mini-tab displays your Production Company information and production start date.  For multiple project productions, the production start date is the date on which the production was created in Lightspeed.  Use the Edit button to edit the Production Company information.

 

Production Company

The Production Company name and address will appear in the header of the Call Sheet and daily Production Report.

 

Preferences (Production)

The Preferences mini-tab allows you to set production-wide options.  For multi-project productions, the Production Preferences offer the following options:

 

·         Whether or not the production will use automatic notifications.  If you uncheck this option, then Lightspeed will not deliver automatic notifications or documents.

·         Document watermarking options

 

Details (Project/Episode)

Select a Project from the list to view the Project's details.  The Details mini-tab displays the project's Status, scheduled dates, and the number of production cast and crew assigned to the project.

 

·         Status

Projects are either Active, Read Only, or Archived.

o       Active Projects are ones available for use.  You can log in and use these Projects as you normally would.

o       Read Only Projects are completed.  You can log into these Projects and view their data, but cannot make any changes.

o       Archived Projects are locked and are not accessible.

 

Click on the Edit button to change a project's name and status.

 

The Default Project

One of the Projects in the Project List is the default Project.  The default Project is the one that users are directed to when they log into the system.  Use the Make Default checkbox to select the viewed project as the new default project.

 

Note: When a project is set to be the default, only those people who are members of that project have their default project changed.  For example, let's say that John Jones is a member of the project Episode 2A but Sam Smith is not.  When Episode 2A is set as the default, the next time John logs in he will be directed to Episode 2A.  Sam's default project will remain unchanged.

 

·         Schedule Start & End Dates

This section lists the project's start date and the projected and revised end dates.  You can edit the Projected End Date in Edit mode.  The start and revised end dates cannot be edited on this screen.  To edit the start date (which represents the first day of principal photography), go to the Calendar edit page.  The revised end date is automatically computed by the production Calendar and strip board.

 

Preferences (Project/Episode)

The Preferences mini-tab allows you to set project-wide options.  These options are only applicable to the selected project, and do not affect any other project in the project list.  These options are:

 

·         Whether or not the project will use automatic notifications.  If you uncheck this option, then Lightspeed will not deliver automatic notifications or documents for this project.

·         Whether or not script page text will be visible online

·         If Hold & Drop rules will be in effect for cast members, and if so, the default number of hold days used

·         The additional Units used on the project

·         Various task assignments for the project.  Assign responsibility for the following production tasks:

o       Creating & publishing Call Sheets

o       Creating & publishing Production Reports

o       Approving Production Reports

 

Individuals assigned production tasks can receive notifications when those tasks are due or overdue.

 

Creating an Additional Unit

Additional Units are created from the Production or Project Preferences screen.  If you are working in a feature film production, go to Production Preferences.  If you are working on an episodic series, go to the Project Preferences for the applicable episode.

 

1.      Select the appropriate Production or Project Preferences mini-tab.

 

2.      Click on the Edit button in the upper right.

 

3.      In the Units table click on the Add Unit button.

A new Unit is added to the list.  You can give the Unit a name up to nine characters long.

 

4.      If the Unit is not actively being used, uncheck the Active checkbox.

 

5.      Click on Save in the lower right corner of the screen to save your changes.

 

Each additional Unit may be assigned its own cast and crew, calendar, strip board within each strip board set, and separate call sheets and PRs.
