Schedule Tab

The Schedule tab provides access to a number of screens related to the production schedule:

 

·         Calendar: The calendar displays the shooting schedule on a monthly grid.  This is also where an administrator sets the starting date of the production phase (the first day of shooting) and selects the days of the week that will be standard working days.  Additional days off are specified here as well.

·         Strip Board: The strip board is used to create the shooting schedule using color coded strips in a vertical list.

·         Strip Board Sets: A strip board set is one particular configuration, or set, of strip boards.  Each strip board set contains a board for each of the following:

o       The main unit schedule

o       Unscheduled strips

o       The Boneyard (strips that have been omitted)

o       A schedule for each additional unit

You can create as many strip board sets as you wish for a production or project.  Each set can be used to schedule the strips a different way.

·         Call Sheet: The current call sheet is available via the Schedule tab.  The call sheet notifies the cast and crew of upcoming call times and scenes.
