Strip Board Sets

A strip board set is one particular configuration of your strip boards.  Each strip board set contains a board for each of the following:

 

·         The main unit schedule

·         Unscheduled strips

·         The boneyard (strips that have been omitted)

·         A schedule for each additional unit

 

You can create as many strip board sets as you wish for a production or project.  Each set can configure the strips in a different way.

 

The strip board sets screen displays all the strip board sets for the current production or episode.  One strip board set, indicated by the In Use column, is the current active set of boards in use at any given time.  The active set is the one that is used to populate the calendar 's shooting schedule with scenes.  The active set determines how many shooting days are in the production.

Strip Boards

When the first script revision is added a strip board is automatically created.  All of the scenes from the script are assigned strips and placed into the unscheduled section.

 

You can perform the following tasks on the strip board set list:

 

Select the Active Strip Board Set

To change which strip board set is actively in use, select the In Use radio button next to the applicable strip board set.  The active strip board change occurs immediately.

 

Edit a Strip Board

Click on the Edit button next to any strip board set to edit it the Main unit schedule for that set of boards.

 

Copy a Strip Board Set

Select the Copy link to create a copy of an existing strip board set.  The copy is immediately created and added to the list.
